# Security Policy

## Supported versions

| Version | Supported |
|---|---|
| latest release | Yes |
| older releases | No |

## Reporting a vulnerability

Please report security vulnerabilities through [GitHub Security Advisories](https://github.com/tldev/shelfrat/security/advisories/new).

Do not open a public issue for security vulnerabilities.

You should receive a response within 72 hours. If the vulnerability is confirmed, a fix will be developed and released as soon as practical.
